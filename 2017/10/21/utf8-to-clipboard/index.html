utf8-to-clipboard

I found a simple way to load any unicode character into clipboard in UTF-8 encoding.
UTF-8 characteristic is that is does encode characters into bytes without necessity to use \x00 byte.

My method to copy “pile of poo” character to clipboard:

function decode_utf8(s) {
return decodeURIComponent(escape(s));
}

sign = decode_utf8(‘\xF0\x9F\x92\xA9’);
copy(sign);
```
