Some AES weakness

AES is asymmetric cipher used to encrypt messages. It can be used in CBC mode and this specific mode will be considered for this post purposes.
Imagine that you know structure of plaintext - which byte of plaintext holds what information. You can leverage this information in order to modify ciphertext (IV actually) to be decrypted to message you want. For example: you know that in plaintext 5th byte is holding boolean 0 or 1 value. To flip this value in ciphertext you intercepted you just need to flip bit in 5th byte of IV. IV is known and can be joined (usually is) together with ciphertext (in front of it).
